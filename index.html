Installation

Quick setup guide to get SimpleShell running on your system.

$ git clone https://github.com/user/simple_shell.git
$ cd simple_shell
$ gcc -Wall -Werror -Wextra -pedantic *.c -o hsh
$ ./hsh

Built-in Commands

Comprehensive list of all built-in commands and their usage.

  • cd [directory] - Change directory
  • env - Display environment variables
  • setenv VAR value - Set environment variable
  • unsetenv VAR - Remove environment variable
  • exit [status] - Exit the shell
  • help [command] - Display help information
